\exercise{Read Chapter 14.}

\exercise{Prove the following using induction for infinite lists on the list $xs$.\ \\


{\bf{Thm. 1}} For every list $ys$ and every infinite list $xs$, $xs = xs\append{}ys$

}

\ \\ \ \\
\noindent{}Consider the definition of the function {\it{cycle}}.
\[\begin{array}{l}
cycle \nil = \bot \\
cycle (x:xs) = (x:xs) \append{} cycle (x:xs)
\end{array}\]



\exercise{Prove the following using induction for infinite lists on the list $xs$.\ \\

{\bf{Thm. 2}} For every infinite list $xs$,  $cycle(xs) = xs$.

}

\ \\ \ \\
\noindent{}Consider the definition of the function {\it{length}}.
\[\begin{array}{l}
length \nil = 0 \\
length (x:xs)  = 1 + length \, xs
\end{array}\]

\exercise{Prove the following using induction for infinite lists on the list $xs$.\ \\

{\bf{Thm. 3}} For every infinite list $xs$,  $length(xs) = \bot$.

}
