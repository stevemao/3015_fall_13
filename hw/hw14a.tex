\exercise{ Write flatten for Rose trees with the following specification:\\
\begin{enumerate}
\item The head of the resulting list is the root of the tree.
\item All of the values in the left most subtree appear first in 
the flattened list.
\end{enumerate}
So for example:
\begin{program*} 
\>   flatten (Node 5 [Node 3 [], Node 7 [Node 8 [], Node 9 []], Node 4 []]) = \\
\>       [5,3,7,8,9,4]
\end{program*}
